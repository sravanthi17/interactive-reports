# Interactive-Reports
Generating Interactive reports with Shiny

**Steps to Setup in Centos**

1) Set up Shiny 

    `sh scripts/setup.sh`
2) Deploy sample (drug count)

    `sh scripts/deployDrugCount.sh`

**Testing**

1) Open the Browser
2) Access the app using the URL  
      
     `http://{ip-address}:3838/sample-apps/`
